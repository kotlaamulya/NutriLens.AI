AI System Workflow

NutriLens uses a YOLOv8 deep learning model trained on IndianFoodNet dataset to detect multiple food items from an image. The detected items are mapped to a nutrition database to calculate calories, proteins, fats and carbohydrates. Additionally, freshness is analyzed using computer vision techniques.


Workflow:

Upload Image → AI Detection → Nutrition Analysis → Recommendation → Freshness Check

About NutriLens

NutriLens is an AI-powered food analysis system designed to help users understand their daily food intake. It combines computer vision and deep learning to detect food items, evaluate nutritional values and provide personalized health insights.


This project is developed as a major academic project focusing on real-world health applications using Artificial Intelligence and Machine Learning.
